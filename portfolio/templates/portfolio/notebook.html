Analysis with open source and proprietary APIs¶

Overview¶

  • The project goal is to analyze and produce summaries of traveling acitivities for members of the sales team in a well documented Jupyter Notebook. Since the data needs to be cleaned and organized, we will implement python libraries to iterate the data, find the desired values, and create an interactive mapping visualization through the ArcGIS API for Python. The steps below document the process to achieve the results in a clear reproducible way.
  • Python libraries utilized / use case:
    • GDAL/OGR -> Creating a shapefile from geometries and attributes stored in a csv.
    • arcgis - ArcGIS API for Python -> Accessing ArcGIS Online to publish the new shapefile and create an interactive map within the notebook.
    • pandas -> Importing / exporting mutiple csv files, iterating, concatenating, and joining data.
    • geopandas -> Importing a csv into a dataframe containing shapely point geometries.
    • os, re, & zipfile -> Working through file expolorer, importing / exporting the desired files, and zipping.

Data Preparation¶

  • List of inputs used for the project from which we can derive our summaries.
    • Employee names
    • Start / End dates of travel time
    • Workspace folder designated as 'projectFiles'
    • Output shapefile
    • Output CSV files

Workflow¶

  • To understand the concept of the workflow lets dicuss the end results. In our final summary we will have a shapefile containing fields for names, ID #'s, duration of travel, and locations connected by a linestring geometry visualized in a map in step 16.
  • Prior to creating this shapefile we will be joining information from a csv containing the employee names and ID #'s to 75 different csv files that contain trip information. The trip information will be combined into one dataframe containing all 75 rows from the individual csv files. We will leverage the power of the pandas API to join and organize our data into one table with the desired fields mentioned above.
  • Once we have moved through the main data processing steps, we will create a shapefile from a csv with the GDAL/OGR API. After using the zipfile module (and getting logged into AGOL with step 14) we can publish the shapefile to our personal Penn State ArcGIS Online account as a feature service. Once step 15 is accomplished we will begin creating the mapping visualization with the ArcGIS API for Python, and then take a look at recreating the data into a different geometry to increase our exerperience with the tools used below.
